[Serverless monitoring for AWS Lambda](/docs/introduction-new-relic-monitoring-aws-lambda) では、Lambda 関数の詳細なパフォーマンスモニタリングが可能です。このドキュメントでは、以下の方法を説明しています。

- [UI で Lambda のデータを探す](#find-data)
- [UI コンポーネントの理解](#ui-pages)
- [チャートデータの把握](#chart-data)
- [カスタムチャートの作成方法](#chart-data)

## 自分のデータを見る [#find-data]

![new-relic-lambda-entities-screenshot.png](src/images/new-relic-lambda-entities-screenshot_0.png 'new-relic-lambda-entities-screenshot.png')

<figcaption>
  **[one.newrelic.com](https://one.newrelic.com/)> Explorer> Amazon Web
  Services> Lambda functions**: Lambda functions
  をクリックするとグラフや詳細が表示されます。
</figcaption>

New Relic で Lambda のデータを見るには： **[one.newrelic.com](https://one.newrelic.com/)** , **エクスプローラーをクリック** in left nav under **Amazon Web Services**, **Lambda functions をクリックします。**

UI の詳細については、 [Intro to New Relic One](/docs/project-wanda-early-access-future-new-relic-platform) をご覧ください。

<Callout variant="important">
  Lambdaのデータが見つからない場合。

- Lambda モニタリングを有効にするための [の指示に従っていることを確認してください](/docs/install-enable-new-relics-monitoring-aws-lambda) 。
- なお、この機能は、 [インフラ監視の Lambda 統合](/docs/integrations/amazon-integrations/aws-integrations-list/aws-lambda-monitoring-integration) とは異なります。
  </Callout>
