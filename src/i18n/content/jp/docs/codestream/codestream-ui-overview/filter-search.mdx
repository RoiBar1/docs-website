New Relic CodeStreamのフィルターと検索ツールを使えば、組織のコメント、課題、フィードバックリクエストのコレクションを自由に切り刻むことができます。デフォルトでは、自分に割り当てられた未解決の課題やフィードバックリクエストが表示され、続いてすべての未解決の課題やフィードバックリクエストが表示されます。

## UIの概要 [#ui]

![フィルターと検索ツールのスクリーンショットです。](/images/FilterAndSearch2.png "フィルター") <figcaption>フィルターと検索ツールを使って、コードストリームの組織を選び、特定のコメント、問題、またはフィードバック要求を見つける。</figcaption>

リストの各項目にカーソルを合わせると、詳細が表示されます。

![ホバーメニューのスクリーンショット](/images/SearchHover1.png "ホバー")

CodeStreamには、強力な定義済みフィルターが多数用意されており、簡単に操作できます。

![あらかじめ設定されているフィルターのスクリーンショット](/images/Filters2.png "フィルター")

また、高度な検索構文を使用して独自の検索を行うことができ、検索ボックスの右端にあるブックマークアイコンから保存して後で使用することができます。

!\[ブックマーク機能を示すスクリーンショットです] (/images/SavedFilter1.png"bookmark")
